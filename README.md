# MILO.ONE – CoRE Theory

CoRE (Control • Regulation • Embodiment) is a theoretical and methodological framework for structured human–AI interaction, with a focus on regulation, embodiment, and dialogic stability.

This repository contains the theoretical foundation, interpretive reports, and ongoing analysis documentation of the CoRE research program.

## Current Status

The project is in an **exploratory-to-confirmatory transition phase**. 
Core analyses (PCA, clustering, MANOVA, CDA, and initial SEM) have been conducted on a corpus of human and LLM-generated texts. Results are documented in the `reports/` and `analysis/decisions/` folders.
